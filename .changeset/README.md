# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, and has been set up to help you manage your project's versioning and publishing.

**New changeset**

To add a new changeset, run `pnpm changeset` and follow the prompts. This will create a new markdown file in this folder that describes the change.

**Releasing**

When you're ready to release, run `pnpm version` to consume all changesets and update package versions. Then run `pnpm release` to publish the packages.

**More info**

See [Changesets documentation](https://github.com/changesets/changesets).
